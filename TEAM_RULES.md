# 🤝 Team Collaboration Protocol

To ensure the project remains stable and both developers work smoothly:

### 🚫 DO NOT:
- Rename folders without updating the mapping in `routes/web.php`.
- Change namespaces without testing the full route list.

### ✅ DO:
- Keep **Frontend Files** in `resources/views/front/`.
- Keep **Backend Logic** in `app/Http/Controllers/Back/`.
- Use the provided guides for each team.
