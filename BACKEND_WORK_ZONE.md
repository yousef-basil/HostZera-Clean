# ⚙️ Backend Development Guide (HostZera)

Welcome! This is your dedicated zone for Logic, Database, and API.

### 📂 Your Main Files:
- **Controllers:** `app/Http/Controllers/Back/`
  - *All backend logic and processing are stored here.*
- **Models:** `app/Models/`
- **Migrations:** `database/migrations/`
- **Routes:** `routes/web.php`

### 💡 Tips:
- Use the namespace `App\Http\Controllers\Back` for any new controllers.
- The Views for the dashboard are located in `resources/views/back/`.
